# China Fund Research Skill

一个用于研究中国市场基金的 Codex Skill，覆盖公募基金、ETF、LOF、QDII、债券基金、指数基金和主动权益基金。

## 能做什么

- 核对基金代码、类型、策略与业绩基准
- 分析收益质量、波动、回撤和持仓集中度
- 检查基金经理、费用、申赎规则与流动性
- 比较多只基金并生成买入前检查清单
- 明确数据日期、缺失信息与主要风险

## 不做什么

- 不替用户下单
- 不承诺收益或预测短期涨跌
- 不使用“稳赚”“必买”等确定性表达
- 不把历史业绩视为未来收益保证

## 安装

将本仓库目录复制到 Codex Skills 目录，并在请求中使用：

```text
$china-fund-research-skill
```
